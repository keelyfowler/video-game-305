Technology

How We Built It

GameVault is built entirely without a custom backend — every service shown below is managed and scalable out of the box.

📱
FlutterFlow
Visual mobile app builder that generates production-ready Flutter code. Enabled rapid UI iteration without writing boilerplate.
🔐
Firebase Auth
Handles secure email/password authentication. Passwords are never stored — Firebase manages tokens and session state entirely.
🗄️
Cloud Firestore
Real-time NoSQL database storing reviews, user profiles, follows, and favorites. Powers the live community feeds with zero latency.
🎮
IGDB API
Twitch's Internet Game Database with 500K+ titles, cover art, genres, platforms, and full descriptions — fetched on demand.
📊
Firebase Analytics
Tracks user engagement, session length, and feature usage. Drives data-informed product decisions throughout the semester.
🔬
Firebase Remote Config
Enables live A/B testing and feature flags without app updates. Lets us experiment on real users and roll back instantly.
🤖
Claude AI (Anthropic)
Powers the AI Recommendations feature — analyzes your favorites list and returns personalized game suggestions via the Claude API.

We validated our design decisions using Firebase Remote Config and Firebase Analytics to run live experiments on real users:

Feed Default Tab
Metric — Day-2 retention & session depth
Variant A — Global (control) Users land on the Global feed by default, seeing all community reviews immediately.
Variant B — Following Users land on the Following feed, nudging them to find friends first. Tested whether social anchoring increased return visits.
Profile Setup Prompt
Metric — Profile completion rate
Variant A — Mandatory (control) Users must fill in display name before accessing the app.
Variant B — Optional with Skip A "Skip for now" button lets users go straight to the feed. Measured whether reducing friction improved Day-1 engagement.
AI Recommendations Entry Point
Metric — AI feature click-through rate
Variant A — Profile page (control) "Ask AI" button lives on the user's own profile screen.
Variant B — Game detail page "Ask AI" button appears contextually on game pages. Tested whether in-context placement drove more recommendations usage.

Business

Lean Canvas

One-page business model capturing the problem, solution, and market for GameVault.

Problem
  • Gamers have no dedicated social app to log and review video games on mobile
  • Existing tools (spreadsheets, Backloggd) lack social features or are web-only
  • Game discovery is fragmented — no single source for peer recommendations
Solution
  • Mobile-first game tracking with star ratings and written reviews
  • Global and social feed for community discovery
  • AI-powered recommendations from your favorites list
Unique Value Proposition
The Letterboxd for gaming. Track, review, and discover video games with your community — free, mobile-first, and powered by 500K+ IGDB titles and AI recommendations.
Unfair Advantage
  • Built by gamers — we are the target user
  • Claude AI integration for personalized recs
  • IGDB's complete, up-to-date game catalog
Customer Segments
  • College-age gamers (18–25) who want to track their library
  • Social gamers who share opinions online
  • Casual players looking for what to play next
Key Metrics
  • Reviews posted per week
  • Daily active users (DAU)
  • AI recommendations tapped
  • Follow network growth
Channels
  • Google Play Store listing
  • URI & college word-of-mouth
  • Gaming Discord communities
  • Social media sharing
Cost Structure
  • Firebase free tier (Firestore, Auth, Hosting, Analytics, Remote Config)
  • IGDB API — free tier, rate-limited
  • Claude AI API — pay-per-call for recommendations
  • Development time — URI CSC 305 student project
Revenue Streams
  • Currently free — no revenue model in v1.0
  • Future: Premium tier with unlimited AI recommendations
  • Future: Affiliate links to game storefronts (Steam, PSN)
  • Future: Optional cosmetic profile upgrades

Privacy Policy

GameVault is committed to your privacy. Here's exactly what we collect, why, and how you can delete it.

Information We Collect

DataPurposeStored?
Email addressAuthenticationYes
Display name & usernameProfile & socialYes
Game ratings & reviewsApp activity & feedYes
Favorites listPersonal log & AI recsYes
Optional bio, Discord, TwitchProfile personalizationIf provided
Camera / photo libraryProfile picture onlyNot stored
Camera access: GameVault does not store camera data. Camera access is used solely to select or capture a profile picture. Raw camera data is never recorded or transmitted beyond that single upload.

How We Use Your Data

  • Create and manage your GameVault account
  • Display your profile, reviews, and favorites in the app
  • Power the global and following community feeds
  • Generate AI-powered game recommendations from your favorites
  • Respond to support requests sent to gamevault@uri.edu

We do not use your data for advertising, profiling, or any purpose beyond operating GameVault.

Data Sharing

  • Firebase (Google) — authentication, database, and file hosting. Governed by Google's Privacy Policy.
  • IGDB (Twitch) — game metadata only. No personal data is sent to IGDB.
  • Public reviews — reviews and ratings you post are visible to all GameVault users by design.

We do not sell, rent, or share personal information with third parties for commercial purposes.

Data Security & Retention

  • All data in transit encrypted via HTTPS / TLS
  • Data at rest encrypted via Firebase's default encryption
  • Passwords are never stored — Firebase Auth manages credentials
  • Data is retained as long as your account is active

While we use industry-standard safeguards, no internet transmission is 100% secure. We cannot guarantee absolute security.

Your Rights & Data Deletion

You have the right to access, correct, or delete your personal data at any time. To request account and data deletion, email gamevault@uri.edu. We will process deletion requests within 30 days.

Public reviews may remain briefly visible after deletion while caches clear.

Children's Privacy & Updates

GameVault is rated Everyone and is not directed at children under 13. We do not knowingly collect personal information from children under 13. If you believe a child has provided information, contact us and we will delete it promptly.

We may update this policy from time to time. Changes will be posted with an updated effective date. Continued use after changes constitutes acceptance.
